GOAL SELECTION STRATEGIES

SELECTION METHODS

Method Description
BY PLAYER EXPERIENCE // New player: Distance goal (100m)
// Intermediate: Mix of Distance and Bonus
// Experienced: All goal types
BY PERFORMANCE // If good at distance but not bonuses:
// Challenge with bonus goal
WEIGHTED RANDOM // Assign weights to each goal type
// Adjust based on player history
VARIETY ENFORCEMENT // Avoid repeating same goal type
// Prioritize unseen types

VALUE DETERMINATION

Method Description
INCREMENTAL Start with base values, increase gradually
PERCENTAGE Increase from player's best (e.g., 120%)
DIFFICULTY CURVE Map player progress to predefined curve

USAGE EXAMPLES

// Show goal message
ShowGoalMessage("DIST_GOAL_ARRAY", 450);

// Show achievement
ShowGoalMessage("GOAL_ACHIEVED_DISTANCE", 450);

// Show progress
string progress = string.Format(GetMessageTemplate("GOAL_PROGRESS_DISTANCE"), 300, 500);
GameMessageDisplay.Instance.ShowMessage(progress);

// Show game status message
GameMessageDisplay.Instance.ShowMessageByID("ZERO_TRIDOTS");
